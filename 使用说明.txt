【软件简介】

C-Clean 是一款智能、安全的C盘清理工具，可以帮助您：
• 扫描并清理临时文件、缓存、日志等
• 检测 Visual C++、.NET、Java 等运行时库
• 智能识别文件所属软件，防止误删
• AI智能分析（需要配置API Key）

【系统要求】

• Windows 10/11（64位）
• 至少 4GB 内存
• 至少 100MB 可用磁盘空间

【使用方法】

下载 C-Clean-1.0.0-portable.exe 后，双击即可运行，无需安装。

【功能说明】

1. 仪表盘
   • 查看C盘使用情况
   • 快速扫描和清理入口
   • 查看扫描统计

2. 文件扫描
   • 扫描临时文件、缓存、日志等
   • 按风险等级分类（安全/谨慎/危险）
   • 选择性清理文件

3. 运行时检测
   • 检测已安装的运行时库
   • 显示依赖该运行时的软件
   • 提供下载链接

4. 设置
   • 切换浅色/深色主题
   • 配置AI API Key
   • 安全选项配置

【AI智能分析配置】

如需使用AI智能分析功能：
1. 打开"设置"页面
2. 选择AI服务提供商（OpenAI或Claude）
3. 输入您的API Key
4. 点击保存

配置完成后，在扫描页面即可使用"AI分析"功能。

【安全说明】

本软件遵循以下安全原则：
• 不会删除系统关键文件
• 清理前会显示风险等级
• 高风险文件需要二次确认
• 支持创建系统还原点
• 记录所有操作日志

【常见问题】

Q: 软件无法启动？
A: 确保系统是 Windows 10/11 64位，并以管理员身份运行

Q: 扫描速度很慢？
A: 首次扫描需要遍历大量文件，请耐心等待

Q: 误删了文件怎么办？
A: 软件会在 AppData 目录记录删除日志，可以尝试恢复

Q: 如何配置AI功能？
A: 在设置页面输入 OpenAI 或 Claude 的 API Key
